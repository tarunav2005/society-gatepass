# 🏢 Society GatePass

A modern MERN Stack Visitor Management System for gated communities that digitizes visitor entry, resident approvals, QR-based gate passes, staff management, notifications, and analytics.

## 🚀 Live Demo

Frontend:
https://society-gatepass.vercel.app

Backend API:
https://society-gatepass-api.onrender.com

✨ Features

- 🔐 JWT Authentication
- 👥 Role Based Access (Admin, Resident, Guard)
- 📱 QR Code Visitor Pass
- 🏢 Tower & Flat Management
- 🚶 Visitor Registration
- 📊 Admin Dashboard
- 🔔 Real-time Notifications (Socket.IO)
- 📄 Excel & PDF Reports
- 🚫 Blacklist Management
- 👷 Staff Management
- 🌙 Dark Mode

## 🛠 Tech Stack

Frontend
- React
- TailwindCSS
- React Router
- Axios

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Socket.IO
